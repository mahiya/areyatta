# 開発するアプリ
- 習慣化アプリ
- 特定のルーチンタスク(以下、タスク)を毎日実行したかどうかをチェックして管理することができる PWA アプリ

# 実装方法
- PWA アプリであり、HTML/CSS/Javascript のみのスタンドアロンで動作する必要がある
  - 登録したタスクとチェック状況は、JavaScript の Local Storage を使用する
  - Local Storage に登録する情報は、JSON 形式でエクスポート/インポートができるようにする
- PWA アプリであるため、Service Worker (sw.js) や manifest.json, またアイコンも作成してください

# タスクの登録
- タスクの登録時には、タイトル, アイコン, 週に何日以上するか？を指定する
- トップ画面では、登録したタスクの一覧と、各タスクの達成状況(過去７日分)を表示する
  - この画面で、特定のエリアを長押しすると、そのタスクを実行したとチェックすることができる
  - チェックした状態で長押しをすると、チェックを解除することができる
- タスク詳細画面では、タスクの達成率や、実行のヒートマップを閲覧できるようにする
